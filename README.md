# homebrew-claude-overlay

Homebrew tap for [claude-overlay](https://github.com/mzmmoazam/claude-overlay).

## Install

```bash
brew tap mzmmoazam/claude-overlay
brew install claude-overlay
```

## About

`claude-overlay` manages project-level Claude Code configuration for custom model providers (Databricks, OpenRouter, LiteLLM, Cloudflare, Bedrock, and more).
